5.23 Object-oriented Forth

Gforth comes with three packages for object-oriented programming: objects.fs, oof.fs, and mini-oof.fs; none of them is preloaded, so you have to include them before use. The most important differences between these packages (and others) are discussed in Comparison with other object models. All packages are written in ANS Forth and can be used with any other ANS Forth.
